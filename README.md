# 🌐 Lazarev Website Clone

A responsive frontend clone of the Lazarev website built using **HTML5**, **CSS3**, and **JavaScript**. This project recreates the modern design, smooth animations, and interactive user interface inspired by the original Lazarev website while strengthening frontend development skills.

## 🚀 Features

* 🎨 Modern and visually appealing UI
* 📱 Fully responsive layout
* ✨ Smooth animations and transitions
* 🖱️ Interactive navigation and hover effects
* 📄 Well-structured and clean code
* ⚡ Fast and lightweight frontend implementation

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 📂 Project Structure

```text
Lazarev-Clone/
│── index.html
│── style.css
│── script.js
│── assets/
└── README.md
```

## 🎯 Purpose

This project was created to practice frontend web development by recreating the design and user experience of the Lazarev website. It focuses on responsive layouts, CSS animations, JavaScript interactions, and writing clean, maintainable code.
